# Airbnb Clone 🏠

A full-stack web application that mimics Airbnb — list, browse, and manage properties.

## Live Demo
Check it out here: [https://airbnb-i209.onrender.com](https://airbnb-i209.onrender.com)

## Features
- User signup/login
- Create, edit, delete property listings
- Upload images
- Search by location
- Responsive design

## Tech Stack
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Frontend:** EJS, HTML, CSS, JS  
- **File Uploads:** Multer + Cloudinary
